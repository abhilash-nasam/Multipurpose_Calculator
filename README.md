# Multi-Purpose Calculator (Python)

A **Multi-Purpose Calculator** built using Python that provides several useful tools in one program.  
This command-line application allows users to perform basic arithmetic, scientific calculations, unit conversions, BMI calculations, and simple interest calculations.

This project is designed for learning and practicing **Python fundamentals such as functions, loops, conditionals, and modular programming**.

---

## Features

### 1. Basic Calculator
Performs standard arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division (with divide-by-zero handling)

### 2. Scientific Calculator
Supports mathematical operations including:
- Square Root
- Power (Exponent)
- Sine
- Cosine
- Tangent

### 3. Unit Converter
Provides simple conversions:
- Kilometer to Miles
- Celsius to Fahrenheit

### 4. BMI Calculator
Calculates **Body Mass Index (BMI)** using:

BMI = weight / (height²)

Inputs:
- Weight (kg)
- Height (meters)

### 5. Simple Interest Calculator
Calculates simple interest using the formula:

SI = (P × R × T) / 100

Where:
- **P** = Principal Amount  
- **R** = Rate of Interest  
- **T** = Time (years)
